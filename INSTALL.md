# 🦇 OwlTeam Opti - Installation Guide

## Requirements
- Windows 10 / Windows 11
- Python 3.8+ (for development)
- 100MB free disk space

## Installation Methods

### Method 1: Download EXE (Recommended)
1. Go to [Releases](https://github.com/MonsterBeetle/owlteam-opti/releases)
2. Download the latest `OwlTeam-Opti.exe`
3. Run the installer
4. Launch the application

### Method 2: Build from Source
1. Clone the repository:
```bash
git clone https://github.com/MonsterBeetle/owlteam-opti.git
cd owlteam-opti
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python main.py
```

4. Or build EXE:
```bash
python build.py
```

## First Run
1. Launch OwlTeam Opti
2. Go to License Manager
3. Your HWID will be generated automatically
4. Enter your license key to activate
5. Start optimizing!
